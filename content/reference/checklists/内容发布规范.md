# 🛡️ 内容发布规范

用于决定哪些页面可公开，并在发布前检查准确性、隐私、版权和链接完整性。

## 发布条件
- 内容经过核验，来源清楚，不包含私人数据或敏感信息。

## Frontmatter
- `publish` 只能由人工确认后修改；`_inbox` 内容不得自动公开。

## 隐私与安全
- 禁止公开 API Key、Token、密码、Cookie、私钥和私人服务器信息。

## 版权与来源
- 引用外部资料时记录来源，不大段复制受版权保护内容。
