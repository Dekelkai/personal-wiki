# 🛡️ 内容发布规范

公开页面应在准确性、隐私、版权和可读性之间保持平衡。

## 准确性与来源

公开的结论应有清楚来源；实验结果、引用和个人判断需要保留各自的边界。

## 隐私与安全

页面不应包含 API Key、Token、密码、Cookie、私钥或私人服务器信息。涉及个人资料、未公开工作或第三方内容时，应先确认公开范围。

## 版权与引用

引用资料时标明来源，并以摘要、评论或必要摘录说明其作用；不大段复制受版权保护的内容。
