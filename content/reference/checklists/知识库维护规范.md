# 📐 知识库维护规范

用于约束日常新增、修改、链接、归档和 Agent 协作行为。

## 新建前检查
- 先搜索标题、别名和近义概念，优先补充已有页面。

## 页面质量
- 正式页面包含完整 Frontmatter、用途说明、实质内容和合理互链。

## 变更安全
- 不直接删除正式笔记，不批量移动目录，实质性重写保留用户原始观点。

## 自动检查
- 修改后运行 `python scripts/check_kb.py` 并处理错误。
