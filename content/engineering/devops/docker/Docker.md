# 📦 Docker

用于整理镜像、容器、数据卷、网络、Compose、构建和故障排查。

## 核心对象
- 镜像描述可复用环境，容器承载运行实例，数据卷保存持久数据。

## 开发流程
- 固化依赖、构建镜像、编排服务并记录版本。

## 安全与维护
- 避免把密钥写入镜像，定期检查基础镜像与权限。
