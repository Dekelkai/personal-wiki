# 🌿 Git

用于整理版本控制、分支、提交、合并、回滚和协作规范。

## 基础工作流
- 小步提交，提交信息说明意图，变更前后检查状态与差异。

## 分支与协作
- 根据项目规模选择主干或功能分支策略。

## 安全边界
- 不提交密钥、Token、密码和私人数据。
