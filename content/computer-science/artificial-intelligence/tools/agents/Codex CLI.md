# ⌨️ Codex CLI

Codex CLI 是面向代码仓库的命令行 Agent，通过项目上下文、权限边界、工具调用和验证命令完成可检查的任务。

## 使用场景
- 代码理解、实现、测试、审查和项目文档维护。

## 上下文组织
- 使用 AGENTS.md、明确范围和可核验的任务描述。

## 安全边界
- 最小权限操作；外部写入、敏感数据和破坏性命令需谨慎确认。
