# 🧠 Claude Code

Claude Code 是面向软件开发任务的命令行 Agent，通过项目上下文、权限控制和工具调用参与代码工作流。

## 项目协作
- 用稳定的项目说明、任务边界和验证命令减少歧义。

## 上下文与记忆
- 区分当前会话信息、项目长期规范和外部资料。

## 风险控制
- 对批量修改、外部命令和不可信内容保持人工审核。
