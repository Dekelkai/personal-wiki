# 🌿 Git

Git 是分布式版本控制系统，通过提交、分支、合并和历史追踪管理代码与文档的变更。

## 基础工作流
- 小步提交，提交信息说明意图，变更前后检查状态与差异。

## 分支与协作
- 根据项目规模选择主干或功能分支策略。

## 安全边界
- 不提交密钥、Token、密码和私人数据。
