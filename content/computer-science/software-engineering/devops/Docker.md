# 📦 Docker

Docker 通过镜像和容器封装应用及其依赖，并通过数据卷、网络和 Compose 管理运行状态与多服务编排。

## 核心对象
- 镜像描述可复用环境，容器承载运行实例，数据卷保存持久数据。

## 开发流程
- 固化依赖、构建镜像、编排服务并记录版本。

## 安全与维护
- 避免把密钥写入镜像，定期检查基础镜像与权限。
