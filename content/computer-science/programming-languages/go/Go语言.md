# 🐹 Go 语言

Go 是静态类型的编译型语言，其标准工具链、并发模型和网络库适合构建命令行工具与网络服务。

## 学习路径

1. [[Go基础语法]]：包、变量、类型、控制流、函数、结构体、接口和错误处理；
2. [[数组与切片]]：固定长度数组、切片长度与容量、底层共享和 `append`；
3. 模块、标准库与项目结构；
4. goroutine、channel、同步和网络服务。

## 工程实践

- 显式处理 `error`，不使用异常隐藏普通失败路径；
- 使用 `gofmt`、`go test`、`go vet` 和模块文件统一代码格式、测试与依赖；
- 算法实现需与输入前提和复杂度一起说明，例如[[二分查找]]。
