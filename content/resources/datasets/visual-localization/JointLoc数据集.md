# 🗃️ JointLoc 数据集

JointLoc 数据集服务于行星无人机视觉定位，主要由仿真火星地形轨迹和少量真实 Ingenuity 图像组成。它与同名 JointLoc 方法和论文关系紧密，但本页面只记录数据集合、真值、协议和使用边界。

## 基本信息

| 项目 | 内容 |
| --- | --- |
| 关联论文 | *JointLoc: A Real-time Visual Localization Framework for Planetary UAVs Based on Joint Relative and Absolute Pose Estimation* |
| 发表 | IROS 2024 |
| arXiv | [2405.07429](https://arxiv.org/abs/2405.07429) |
| 数据性质 | 行星地形仿真轨迹，附少量真实火星图像及配准合成图 |
| 主要任务 | 绝对 2-DoF 位置、相对 6-DoF 运动和世界坐标轨迹融合 |

- 项目仓库：[LuoXubo/JointLoc](https://github.com/LuoXubo/JointLoc)
- 论文：[arXiv:2405.07429](https://arxiv.org/pdf/2405.07429)

GitHub 仓库显示 MIT 许可证，但该许可证只能确认代码许可。截至 2026-07-16，项目页没有明确说明测试数据的独立许可证，因此数据许可记录为“待验证”。

## 版本与下载状态

官方仓库在论文接收后上传了 test dataset，并提供安装、目录结构和运行说明。截至 2026-07-16，仓库没有正式 Release，数据版本主要通过项目页更新记录识别。

下载和实验前应记录：

- 获取日期；
- Google Drive 文件名与校验值；
- 轨迹目录数量；
- 图像、卫星图、真值文件和配置是否完整；
- 数据许可或作者附加说明。

## 数据性质与采集方式

主要轨迹通过仿真引擎生成，覆盖三类典型行星地形。论文还使用 NASA 公开的真实 Ingenuity 图像，并通过人工配准构造对应的合成图像。

真实图像部分不能视为真实火星飞行中的完整同步视频、IMU 和世界坐标 6-DoF 真值数据。

## 数据规模

- 6 组仿真轨迹；
- 3 类地形：砾石、山地和陨石坑；
- 3,000 张主要仿真 UAV 视角图像；
- 105 张真实 Ingenuity 图像；
- 95 张人工配准的合成图像。

3,000 张轨迹图像与 105+95 张真实—合成配准图属于不同用途，不应相加为一个没有统计口径的统一规模。

## 传感器、地图与坐标

数据目录包括 UAV 图像、卫星或地形参考图、图像索引和轨迹真值文件。论文中的定位任务将地图匹配得到的世界坐标二维位置与视觉里程计产生的相对 6-DoF 位姿结合。

具体坐标轴、单位、姿态表示和时间戳格式需要在下载后结合 `gt.txt`、`images.txt` 和运行配置继续核验。

## 真值与标注

仿真轨迹提供用于世界坐标轨迹评价的真值。真实 Ingenuity 图像部分主要用于跨模态图像匹配测试，并没有构成与仿真轨迹等价的完整传感器真值序列。

论文方法输出世界坐标 6-DoF 轨迹，但这是绝对位置与相对运动融合后的任务输出，不意味着每张真实火星图像都具有直接测得的完整 6-DoF 标签。

## 数据划分和评价协议

项目仓库将公开内容称为 test dataset，没有给出稳定的训练、验证和测试划分规范。

仓库示例使用 `evo_ape` 比较估计轨迹和参考轨迹。论文还使用位置误差均值、RMSE、标准差、最小值、最大值和中位数，以及运行速度等统计。

论文报告的数值属于特定方法、轨迹和配置下的作者实验结果，不能写成数据集固有性能，也不能表述为用户已经复现。

## 适用任务

- 行星弱纹理环境中的 UAV—地图匹配；
- 绝对二维位置与相对 6-DoF 运动融合；
- 世界坐标连续轨迹估计；
- 合成火星地形上的定位和漂移研究；
- 少量真实火星图像与合成地图的跨模态匹配。

## 已知限制与版本差异

- 主要轨迹是仿真数据，与真实火星成像、噪声和地貌统计存在域差异；
- 真实图像数量少，且不包含完整同步传感器和高精度轨迹真值；
- 数据许可证未明确；
- 未提供稳定的训练、验证和测试划分；
- JointLoc 同时是方法、论文和数据集合名称，引用时需要说明对象；
- 本页不包含用户实际下载或复现实验结果。

## 与相关主题的关系

JointLoc 数据集属于行星和弱纹理场景的相邻研究分支，用于检验绝对地图定位与相对运动融合思路。[[AerialVL]] 和 [[OrthoLoC]] 则分别补充地球场景中的连续航线和几何 6-DoF 定位证据。
