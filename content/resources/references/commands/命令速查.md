# ⌨️ 命令速查

命令速查按平台和任务整理经过验证的常用命令，并标注适用环境、前置条件和风险。

## 系统命令
- Linux、PowerShell 与 Windows 系统管理命令。

## 开发命令
- Git、Docker、Go、Python 和网络诊断命令。

## 安全要求
- 破坏性命令必须说明影响范围，不记录真实凭据和私人地址。
